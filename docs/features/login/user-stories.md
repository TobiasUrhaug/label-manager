# User Stories — Login

## US-01: Log in to the application
As a label manager,
I want to log in with my username and password,
So that I can access my label's data securely.

## US-02: See feedback on a failed login
As a label manager,
I want to see an error message when I enter incorrect credentials,
So that I know my login attempt failed and can try again.

## US-03: Log out of the application
As a label manager,
I want to log out,
So that my session is ended and my data is protected.

## US-04: Be redirected to login when not authenticated
As a label manager,
I want to be redirected to the login page when my session has expired or is absent,
So that I am prompted to authenticate before seeing any data.
